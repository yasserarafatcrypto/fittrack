🔗 Connect Google Fit

To auto-sync your weight from Omron → Google Fit, you need a free Google Cloud API key (one-time, ~5 min setup).

1Open console.cloud.google.com
2Create a new project (any name)
3APIs & Services → Enable Fitness API
4Credentials → Create → OAuth 2.0 Client ID → Web Application
5Add your app URL to Authorised JavaScript Origins

❤️ Connect Fitbit

Syncs RHR and HRV automatically from your Fitbit. One-time setup (~5 min) at dev.fitbit.com.

1Go to dev.fitbit.com/apps/new
2Fill in any app name. Set OAuth 2.0 Application Type → Client
3Set Callback URL to your GitHub Pages app URL
4Copy your OAuth 2.0 Client ID (short code like 23ABCD)
